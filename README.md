# pitch-odds
Football quantitative analysis system — probability modeling, odds anomaly detection, and parlay optimization for the top 5 European leagues and FIFA World Cup 2026.

## Phase 1 quick start

```bash
pip install -r requirements.txt
bash scripts/run_phase1_e2e.sh "/path/to/E0_2024_25.csv" --league-id E0 --season 2024-25 --bookmaker bet365
```

详细测试步骤见 `docs/phase1_test_checklist.md`。
